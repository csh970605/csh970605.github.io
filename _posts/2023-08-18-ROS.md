# What is ROS?

ROS provides various libraries and tools for implementation and debugging.

## Functions that ROS Provides
---
<br>

+ Hardware abstraction

+ Low-level device control

+ Function implementation
    
    + Sensing

    + Detecting

    + Map making(SLAM)

    + Motion planning

+ Message passing between processes

+ Package management
